## 格式规范

| 字段 | 说明 |
|------|------|
| 📅 标题行 | 固定格式，包含日期时间 |
| 热榜摘要 | 显示数据来源和条数，列出 Top3 最热话题 |
| 账号定制标注 | 括号内填用户账号 niche 和目标受众 |
| 选题分类 | 三类：热点蹭榜 / 趋势借势 / 原创高潜 |
| 每条选题 | 标题 + 热点关联（如有）+ 匹配理由 + 拍摄建议 |
| 总结 | 一句话概括今日内容方向建议 |
| 底部提示 | 固定显示更新偏好的操作指引 |

## 选题标题写法原则

- 用数字：《月薪5000的我，找到了这 5 家不排队的宝藏餐厅》
- 制造悬念：《这家店被差评淹没，但我吃完改变了想法》
- 强对比：《人均 8 元 vs 人均 80 元，差距到底在哪？》
- 情绪共鸣：《下班后最治愈的事，就是一个人吃顿好的》
- 实用导向：《不踩坑指南：这 3 类网红店，我劝你别去》
